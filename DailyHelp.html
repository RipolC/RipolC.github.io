Este proyecto fue desarrollado como trabajo final tras completar mis estudios de Grado Superior. Se trata de una aplicación móvil diseñada para gestionar eventos personales en un calendario, integrando funcionalidades avanzadas que van más allá de una simple agenda.

  • Gestión completa de eventos: Permite crear, visualizar y eliminar eventos asociados a fechas y horas específicas, con notificaciones automáticas que alertan al usuario cuando se aproxima la hora del evento.
  • Sincronización con Outlook: La aplicación permite iniciar sesión con una cuenta de Outlook mediante el protocolo de autenticación OAuth2 (implementado con MSAL) y sincronizar automáticamente los eventos del calendario online con los eventos locales de la app, facilitando una experiencia unificada para el usuario.
  • Asistencia mediante inteligencia artificial: Gracias a la integración con una API de ChatGPT, la aplicación puede ofrecer consejos personalizados en función del clima del día. Para ello, consulta automáticamente una API de pronóstico del tiempo (WeatherAPI) usando la ubicación del dispositivo, y genera un mensaje útil o motivacional para el usuario.

Este proyecto combina conocimientos de desarrollo móvil en Android (Java y XML), bases de datos locales con SQLite, integración de APIs externas, autenticación segura y uso de inteligencia artificial.
