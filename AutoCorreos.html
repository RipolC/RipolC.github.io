AutoCorreos

Durante mi estancia en AST desarrollé mi primer proyecto de software completo: una aplicación de escritorio escrita en Java, diseñada específicamente para automatizar procesos rutinarios y mejorar la eficiencia operativa del departamento de Recursos Humanos. Esta aplicación fue implementada utilizando NetBeans IDE, y hace uso intensivo de Java Swing para la interfaz gráfica, JDBC para la conexión a una base de datos SQL, y bibliotecas externas para la integración con Microsoft Outlook y la generación de archivos PDF.

El sistema automatiza tareas clave como:

  • Gestión y mantenimiento de la base de datos de empleados, permitiendo realizar operaciones CRUD (crear, leer, actualizar y eliminar) sobre registros con información personal, historial de formación y estado laboral.
  • Generación automática de certificados en formato PDF para los empleados que completan formaciones internas o externas, incluyendo detalles como el nombre del curso, fechas y duración.
  • Envío automatizado de certificados por correo electrónico, eliminando la necesidad de distribuirlos manualmente, lo cual agiliza significativamente la comunicación interna.
  • Programación automática de reuniones en Outlook, mediante la creación de eventos calendarizados y el envío de invitaciones a los participantes según los registros de formación o asignación de tareas.

El proyecto fue desarrollado con una arquitectura modular, separando claramente la interfaz de usuario, la lógica de negocio y el acceso a datos. Se implementaron controles de validación y un manejo robusto de errores. La experiencia incluyó también la integración con bibliotecas de terceros para la manipulación de calendarios de Outlook y la creación de documentos PDF.
