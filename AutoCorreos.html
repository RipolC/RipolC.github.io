AutoCorreos

Durante mi estancia en AST desarrollé mi primer proyecto de software completo: una aplicación de escritorio en Java orientada a automatizar procesos repetitivos del departamento de Recursos Humanos. Esta herramienta facilita la gestión de una base de datos SQL con información de empleados, permite el envío automatizado de reuniones a través de Outlook, y genera certificados para aquellos trabajadores que completan formaciones internas o externas. Por motivos de confidencialidad, no se incluyen capturas ni código fuente del proyecto.
